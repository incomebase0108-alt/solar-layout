# ☀ ソーラーレイアウト設計支援

既設太陽光発電のレイアウト図面化と、パネル入換時の最適化・ストリング設計（直列/並列）・配線図作成を支援する Web アプリです。

## 目的（全体像）

1. **現況図の作成** — 航空写真を背景に手動トレース＋基準寸法でスケールを確定し、現在のパネル配置を図面化
2. **入換レイアウトの最適化** — 全交換／一部交換を想定し、「同サイズ〜少し大きいサイズ」で収まる最適なパネル割付を算出
3. **配線図の作成** — パワコンを既設流用／新設から選択し、PCS 仕様に合わせて直列数・並列数を決めた配線図を出力（同一系統＝同一パネル）
4. **マスタ登録** — パネル登録・パワコン登録画面から直列/並列を計算

## 開発

```bash
npm install
npm run dev      # 開発サーバ
npm test         # 計算ロジックのテスト
npm run build    # 本番ビルド
```

データはブラウザの localStorage に保存されます（マスタ・設計条件）。

## 計算ロジックの考え方

- **最大直列数**：低温時 Voc（温度補正）× 直列数 が PCS 最大入力電圧を超えない範囲
- **最小直列数**：高温時 Vmp（温度補正）× 直列数 が MPPT 動作電圧下限（・起動電圧）を下回らない範囲
- **最大並列数**：PCS の MPPT あたり最大入力電流 ÷ パネル Isc と、ハード上限（最大並列数）の小さい方
- **過積載率**：DC 容量 ÷ PCS 定格（AC）
